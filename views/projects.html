Bees

They're good workers.

Each bee has an artificial neural network for a brain. The network takes in information about the bee's surrounding as input. Bees have five eyes: one in front and two equally spaced on either side. A bee can see another object if it crosses one of its lines of sight. The bee can then see what type of object that thing is, how far away it is, and what direction it is headed (if any). The outputs of the network tell the bee what action to take. A bee can speed up, slow down, turn, grab food, drop food, or attack. These brains were trained over thousands of generations in a genetic algorithm.

The queen takes food and creates new bees. These bees can be either workers or fighters. Workers go and collect more food and bring it back to the queen. Fighters attack other bees that are not related.

This project was originally coded in Ruby (specifically in jRuby), and I'm working on making a version in JavaScript so it can be shown here.
